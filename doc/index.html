wixgem

Ruby gem to facilitate automate constructing Windows installation files with the Wix Toolset.

Installation

wixgem can be installed by the single command gem install wixgem

Usage

The wix toolset must be installed.

Simple usage

Dependencies

Wixgem requires the WiX Toolset to be installed.

Installation

require 'wixgem'
	
WIX_TOOLSET_ROOT='path to root of Wix toolset'
Wix.make_installation('Product.msi', ['rakefile.rb']])

Merge Module

require 'wixgem'
	
WIX_TOOLSET_ROOT='path to root of Wix toolset'
Wix.make_mergemodule('Product.msi', ['rakefile.rb']])

For an example rakefile click [example/rakefile.rb]
